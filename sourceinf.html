Shell-like命令
Commands on Shell-like

  • mkdocs new [dir-name]

创建一个新的mkdocs项目。

Create a new project.

  • mkdocs serve

在主机上于pwd启动mkdocs服务，默认的网址为http://127.0.0.1:8000/，这因人而异，但是在Shell-like上会显示出来

Start the live-reloading docs server. The default URL is http://127.0.0.1:8000/, which differs for each computer but will be printed out on Shell-like command end.

  • mkdocs build

构建网站。

Build the documentation site.

  • mkdocs -h

输出帮助信息。

Print help message and exit.
